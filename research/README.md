## How research is used in this repo

Three rules:

1. **Every mechanistic claim links to a citation.** If you find a claim in a case study or protocol without a citation, it's an error — open an issue.
2. **n=1 personal experience is labeled as such.** Citations support general mechanisms; personal data supports personal claims.
3. **⚠️ flags mean verify.** A few citations in this repo are from author recall and have authors/journal/year right but the exact PMID needs PubMed confirmation. These are flagged.

## How to verify a citation

For any cited paper:

1. Copy the title or first-author + year into PubMed: https://pubmed.ncbi.nlm.nih.gov/
2. Confirm authors, journal, year, and volume/page match what's in this repo
3. If anything is wrong, open a GitHub issue with the correct citation
